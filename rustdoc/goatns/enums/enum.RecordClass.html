Enum goatns::enums::RecordClass

source · 
pub enum RecordClass {
    Internet,
    CsNet,
    Chaos,
    Hesiod,
    InvalidType,
}
Expand description

CLASS fields appear in resource records, most entries should be IN, but CHAOS is typically used for management-layer things. Ref RFC1035 3.2.4.
